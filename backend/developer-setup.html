Setup Instructions

Windows Instructions:
  1. Download both sync.ps1 and start_sync.bat files
  2. Save them to a folder like C:\ActivityWatchSync\
  3. Double-click start_sync.bat to start syncing
  4. For auto-start: Press Win + R, type shell:startup, copy the bat file there
Mac/Linux Instructions:
  1. Download sync.sh
  2. Open Terminal and navigate to download folder
  3. Make it executable: chmod +x sync.sh
  4. Run it: ./sync.sh
  5. For auto-start, add to your system's startup applications
Verification:
The sync will run every 5 minutes and send your ActivityWatch data to the server automatically.
You should see confirmation messages in the console window.
